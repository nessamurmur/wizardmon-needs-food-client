# Wizard Needs Food

A Web Worker (Maybe soon, Service Worker) demo.

## Development

``` bash
npm install -g typings
npm install
typings install
npm run dev-server
```

## Test

``` bash
npm test
```

## Build

```bash
npm run build
```
